Interactive multivariate learning tool: PCA, distance, clustering, MDS, models

This keeps the original raw-value PCA vs rank PCA app and expands it into a broader syllabus tool. You can compare scaled raw values with ranked values, then move into distance structure, MDS, clustering, supervised models, validation, factor analysis, and a small correspondence analysis demo.

Controls

Start with PCA core, then move to Distance Explorer and MDS to see geometry, then use Clustering and Supervised Models to test whether the same structure is useful for grouping and prediction.

Distance explorer

Classical MDS

Classical multidimensional scaling starts from a distance matrix and places points so that distances in 2D stay as faithful as possible. PCA starts from variables and variance. MDS starts from distances and geometry.
