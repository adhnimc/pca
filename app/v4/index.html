Interactive PCA: scaled raw values vs ranked values

Yes — the earlier version used simulated random data. This version lets you switch between simulated data and a few real built-in datasets. It compares PCA on scaled raw values with PCA on ranked values (Spearman-style PCA), then shows score agreement, biplots, and Pearson vs Spearman correlation heatmaps.

Controls

Simulated mode creates random data with one hidden driver, noise, nonlinearity, and optional outliers.
For real datasets, the simulation sliders are ignored. If a real dataset has class labels, the points are colored by class so you can see whether raw PCA and rank PCA preserve the same structure.
