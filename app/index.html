PCA: raw values vs rank PCA

Repository bootstrap committed from ChatGPT.

This repo is for an interactive project comparing:

  • PCA on scaled raw values
  • PCA on ranked values (Spearman-style PCA)

The fuller interactive prototype currently exists in the local working artifact and can be moved into this repo in the next commit/refactor.

Planned features

  • Simulated dataset with controls for noise, nonlinearity, and outliers
  • Real dataset mode
  • PC1 and PC2 score comparisons
  • Raw PCA and rank PCA biplots
  • Pearson vs Spearman correlation heatmaps
  • Loadings table
  • CSV upload

How to run later

cd app
python3 -m http.server 8000

Then open http://localhost:8000.
